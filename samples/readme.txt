This directory contains Cornell's 29 sample records converted to RDF, both the 
output from the LC converter and the versions manually converted from these to 
the LD4L proposed ontology.

LC converter version: master revision 8fbb247e
Conversion date: 2015-06-01


Filenames:

bibid.original.rdf - e.g., 102063.original.rdf
bibid.revised.rdf - e.g., 102063.revised.rdf

URIs minted in namespace http://ld4l.library.cornell.edu/ld4l-vitro/individual/ 
for ingest into a demonstration Vitro instance. This can be changed for public
distribution if desired.
